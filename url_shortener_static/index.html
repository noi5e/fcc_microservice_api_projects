URL Shortener Microservice

User stories:
  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://fcc-api-projects-noi5e.c9users.io/url-shortener/https://www.google.com

Example creation output:

{ "original_url":"https://www.google.com", "short_url":"https://fcc-api-projects-noi5e.c9users.io/url-shortener/4" }

Example redirection usage:

https://fcc-api-projects-noi5e.c9users.io/url-shortener/4/

Example redirection output:

Redirects to: https://www.google.com
